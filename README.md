# Simple-TicTacToe-Python
This is a Simple Tic-Tac-Toe game built using **_Python_**.

## Dependencies Required

Python3

### Install Python3
**mac**
```bash
brew install python3
```
**Debian/Ubuntu**
```bash
sudo apt update
sudo apt install python3
``` 

## Usage
```bash
python3 TicTacTie.py
```

**ENJOY THE GAME**
